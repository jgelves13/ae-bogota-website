## Common Patterns Across All Pages

### Card Component
- **Structure**: Image (top) → Heading (Inter, bold) → Description (Times New Roman, regular)
- **Styling**: No border, no shadow, no rounded corners, no background color (sits on `#f4efe9`)
- **Image**: Rectangular, no border-radius, varies between roughly 3:2 and 16:9 ratios
- **Grid**: Always 3 columns on desktop, with occasional 2-item rows left-aligned

### Section Pattern
- Every section follows: Heading → Optional subtitle → Content grid
- Generous vertical padding between sections (~60-90px)
- Content max-width constrained to ~1240px, centered
- Headings are always Inter, bold, with negative letter-spacing

### Accordion Component (FAQ/Objections)
- Full-width rows with question text + "+" icon
- Separated by thin black horizontal lines
- No background change, no hover effects visible
- Very minimal and text-focused

### Buttons
- Primary: black bg, white text, square corners (0px radius), thin border
- Outlined: transparent bg, black border, black text, square corners
- No shadows, no gradients, no hover color change visible

### Links
- Blue (`#0000ee`) for standard links
- Teal (`#0c859a`) in some article contexts
- Underlined by default in body text

### Overall Feel
- The site reads like a digital magazine or journal
- Heavy use of whitespace and the warm `#f4efe9` paper-like background
- Photography is real, editorial, human-centered (not stock-photo-feeling)
- Typography contrast between serif body and sans-serif headings is the defining visual trait
- Almost no decorative elements — no icons (except social), no illustrations, no gradients, no patterns
