# EA Bogotá — Style Guide (from effectivealtruism.org)

## Colors
- Page background: `#F4EFE9` (rgb 244, 239, 233) — warm sand, NEVER white
- Text primary: `#000000` — pure black
- Text muted: varies by context
- Teal accent: `#0C859A`
- Footer background: `#000000` — pure black
- Card background: none (flat on sand)

## Typography

### Headings — Inter (sans-serif), weight 600, NEGATIVE letter-spacing
| Level | Size | Weight | Letter-spacing | Line-height |
|-------|------|--------|---------------|-------------|
| h1    | 54px | 600    | -2px          | 1.05        |
| h2    | 30px | 600    | -1.2px        | 1.3         |
| h3    | 22px | 600    | -0.28px       | 1.2         |
| h4    | 20px | 600    | -0.28px       | 1.2         |
| h5    | 18px | 600    | -0.2px        | 1.2         |
| h6    | 16px | 500    | -0.2px        | 1.2         |

### Body — Times New Roman (serif), weight 400
- Font: "Times New Roman", serif
- Size: 16px
- Weight: 400
- Line-height: normal (~1.5)

## Buttons
- Background: `#000000`
- Text: `#FFFFFF`
- Border-radius: `0px` (square)
- Border: `1px solid`
- Padding: `12px 24px`
- No box-shadow
- Font: Inter, 16px, weight 400

## Cards
- No border
- No shadow
- No background (flat on `#F4EFE9`)
- Sit directly on the page background

## Layout
- Max content width: `1280px`, centered
- Section vertical padding: 48-90px
- Grid gap: ~24px
- 3-column grids for card layouts

## Footer
- Background: `#000000`
- Text: `#FFFFFF`
- Padding: 32px 24px

## General Rules
- No box-shadows anywhere
- No gradients
- No rounded buttons
- Editorial/journal aesthetic — warm, flat, typographically driven
- Lots of whitespace
- Serif body + sans-serif headings
